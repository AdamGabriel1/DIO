# Projeto: Utilizando Prompts para Gerar Insights de Relatórios de Vendas

## Objetivo do Projeto
Desenvolver um sistema que utilize prompts para gerar insights a partir de dados de vendas, melhorando a análise e a tomada de decisões.

## Estrutura do Projeto

### 1. Introdução
- **Descrição do Projeto**: Apresentar a importância de utilizar prompts para gerar insights a partir de relatórios de vendas.
- **Metodologia**: Explicar como os prompts ajudarão na análise de dados e na geração de relatórios.

### 2. Pré-requisitos
- **Conhecimentos Necessários**:
  - Noções básicas de Python.
  - Familiaridade com análise de dados.
  - Conhecimento em ferramentas de visualização (ex: Tableau, Power BI).

### 3. Use Case
- **Cenário**: Descrever um caso de uso específico, como a análise de vendas mensais para identificar tendências.
- **Objetivos**: O que se espera alcançar com a análise (ex: aumento de vendas, melhor entendimento do cliente).

## Exemplos de Prompts para Análise

1. **Análise de Vendas**:
   - "Quais foram os cinco produtos mais vendidos no último trimestre?"
   
2. **Tendências de Mercado**:
   - "Como as vendas mudaram em relação ao ano passado?"

3. **Perfil do Cliente**:
   - "Qual é a idade média dos nossos clientes que compraram mais de uma vez?"

4. **Desempenho Regional**:
   - "Quais regiões tiveram o melhor desempenho em vendas no último mês?"

Esse projeto fornece uma estrutura clara para utilizar prompts na análise de dados de vendas. Com a prática, você poderá gerar insights valiosos que podem impactar diretamente nas estratégias de vendas e marketing.
